# Blinkit-Sales-Analysis-Dashboard
The Blinkit Sales Analysis Dashboard is an interactive Power BI project designed to analyze sales performance, customer preferences, outlet characteristics, and product trends across Blinkit's retail network. The dashboard transforms raw sales data into actionable business insights, enabling stakeholders to monitor key performance indicators and make data-driven decision.

# Dashboard Highlights
- Key Performance Indicators (KPIs)
- Total Sales: $1.20M
- Average Sales: $141
- Number of Items Sold: 8,523
- Average Customer Rating: 3.9

# Business Insights
- Outlet Establishment Trend: Sales peaked at $205K for outlets established in 2018.
- Outlet Size Analysis: Medium-sized outlets generated the highest sales contribution.
- Outlet Location Analysis: Tier 3 locations achieved the highest sales performance ($472.13K).
- Fat Content Analysis: Regular-fat products contributed more sales compared to low-fat products.
- Item Type Performance: Fruits & Vegetables and Snack Foods emerged as the top-selling categories.
- Outlet Type Comparison: Supermarket Type 1 generated the highest revenue and item sales volume.
