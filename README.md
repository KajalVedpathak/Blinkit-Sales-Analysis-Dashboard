# Blinkit-Sales-Analysis-Dashboard
This project presents an interactive Power BI dashboard built to analyze Blinkit's sales performance, customer behavior, and product trends. The dashboard provides actionable business insights through KPIs, sales trends, outlet performance, product category analysis, and customer purchasing patterns. It enables data-driven decision-making by identifying top-performing products, revenue drivers, and opportunities for business growth.

# Key Insights
Total Sales, Average Sales, and Number of Items Sold
Sales Performance by Outlet Type and Location
Fat Content and Item Category Analysis
Outlet Establishment Trends
Customer Purchase Patterns
Interactive Filters and Drill-Down Analysis

# Tools Used
Power BI
Excel / CSV
DAX
Data Modeling
Data Visualization

# Business Value
Identifies high-performing products and outlets
Reveals customer purchasing trends
Supports data-driven decision-making
Highlights opportunities for sales optimization
